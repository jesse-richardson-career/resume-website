About This Site

This is where the magic happens. I appreciate websites with clean designs, minimal information with maximum clarity, and a bit of color to make things pop.

This site was made with those ideas in mind. I built this website from scratch using basic HTML, JavaScript, and CSS/SASS, as well as tools like jQuery, Bootstrap, and Font Awesome. In creating this site, I used responsive design principles, so it doesn't just look nice on wide screens, but on mobile phones as well, and everything in between.
